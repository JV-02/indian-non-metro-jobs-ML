Analyze job market trends and predict salary/opportunity patterns using regression techniques.

Python, Pandas, Scikit-learn

- Exploratory Data Analysis
- Feature selection
- Linear Regression, Decision Tree, Random Forest
- Model comparison

## Key Learnings
- Regression modeling
- Model evaluation
- Real-world job market analysis

## Files
- jobs_ml_project.ipynb

## How to Run
Execute notebook cells in order to train and evaluate models.
